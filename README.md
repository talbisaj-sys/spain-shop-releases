# Spain Shop - Application Mobile

Boutique de vêtements en ligne connectée à Appwrite.

## 📱 Téléchargement Direct

| Architecture | Compatible avec | Téléchargement |
|-------------|---------------|----------------|
| **ARM 64-bit** ⭐ | Smartphones modernes (Samsung, Xiaomi, Huawei 2017+) | [📥 Télécharger APK](https://github.com/talbisaj-sys/spain-shop-releases/raw/main/releases/v1.0.0/app-arm64-v8a-release.apk) |
| **ARM 32-bit** | Anciens smartphones (Android &lt; 6.0) | [📥 Télécharger APK](https://github.com/talbisaj-sys/spain-shop-releases/raw/main/releases/v1.0.0/app-armeabi-v7a-release.apk) |
| **x86_64** | Émulateurs, tablettes rares | [📥 Télécharger APK](https://github.com/talbisaj-sys/spain-shop-releases/raw/main/releases/v1.0.0/app-x86_64-release.apk) |

&gt; 💡 **Conseil** : La plupart des téléphones récents utilisent l'architecture **ARM 64-bit**.

---

## 🔧 Installation

1. **Télécharge** l'APK adaptée à ton appareil (tableau ci-dessus)
2. **Autorise** les sources inconnues : `Paramètres → Sécurité → Sources inconnues`
3. **Installe** l'APK
4. **Ouvre** l'application Spain Shop

---

## 🌐 Fonctionnalités

- ✅ Catalogue de vêtements
- ✅ Panier d'achat
- ✅ Commandes en ligne
